# PcMqttAgent 🖥️📡

Легковесное фоновое приложение для Windows 10/11, работающее в системном трее. Предназначено для мониторинга состояния ПК (загрузка CPU/RAM, температуры, uptime) и удаленного управления (выключение, перезагрузка) через протокол **MQTT**.

Идеально подходит для интеграции с системами умного дома (Home Assistant, Node-RED, MajorDoMo) или для удаленного администрирования.

## ✨ Возможности

- 🖥️ **Работа в фоне**: Скрытое окно, иконка в системном трее с контекстным меню.
- 📊 **Мониторинг железа**: Загрузка CPU, RAM, время работы (Uptime) и температуры (через `LibreHardwareMonitor`).
- 📡 **MQTT интеграция**: 
  - Автоматическое переподключение при обрыве связи.
  - Поддержка **Last Will and Testament (LWT)** для мгновенного обнаружения оффлайн-статуса при краше или отключении питания.
  - Публикация данных с настраиваемым интервалом.
- 🎮 **Удаленное управление**: Выполнение системных команд (Shutdown, Reboot) через MQTT-топики.
- ⚙️ **Гибкая настройка**: Все параметры хранятся в `appsettings.json`.
- 📝 **Логирование**: Подробные логи с ротацией по дням через `Serilog`.

## 📋 Требования

- **ОС**: Windows 10 / Windows 11
- **.NET**: .NET 9.0 Runtime (или .NET 9.0 SDK для сборки)
- **Права**: Для чтения температур и выполнения команд выключения **требуется запуск от имени Администратора**.

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/ВАШ_НИК/PcMqttAgent.git
cd PcMqttAgent
2. Сборка и запуск
Убедитесь, что терминал (или Visual Studio Code) запущен от имени Администратора, затем выполните:
dotnet restore
dotnet build
dotnet run
⚙️ Настройка
Откройте файл appsettings.json и укажите параметры вашего MQTT-брокера:
{
  "Mqtt": {
    "Server": "mqtt-server.lan",
    "Port": 1883,
    "ClientId": "MAINPC",
    "User": "your_username",
    "Password": "your_password",
    "BaseTopic": "MAINPC"
  },
  "Publisher": {
    "IntervalSeconds": 60
  },
  "Logging": {
    "FilePath": "logs/agent-.log",
    "LogLevel": "Information"
  }
}
📡 Структура MQTT-топиков

Откройте Планировщик заданий (taskschd.msc).
Нажмите "Создать задачу..." (не "Создать простую задачу").

⚠️ Известные ограничения оборудования
Температура CPU на мобильных AMD Ryzen (например, 5500U): Из-за ограничений драйверов и BIOS некоторых ноутбуков/мини-ПК, датчик Core (Tctl/Tdie) может постоянно возвращать 0°C. Это ограничение "железа", а не приложения.
Температура встроенной графики (iGPU): Встроенные видеокарты (AMD Radeon Graphics, Intel UHD) обычно не имеют выделенных аппаратных датчиков температуры, поэтому MAINPC/info/temp/gpu будет возвращать N/A.
